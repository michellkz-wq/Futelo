Le test couvre les principales technologies haut débit : fibre FTTH (fibre jusqu'au logement), câble FTTLA (réseau câblé hybride), ADSL et VDSL. Pour les box 4G/5G, aucun test d'éligibilité n'est nécessaire — elles fonctionnent partout où il y a du réseau mobile.
Si la fibre n'est pas disponible à votre adresse, plusieurs alternatives existent : le câble (SFR) si vous êtes en zone câblée, la box 4G/5G si vous avez une bonne couverture réseau mobile, ou l'ADSL/VDSL en attendant le déploiement de la fibre. Consultez notre comparateur box 4G/5G.
